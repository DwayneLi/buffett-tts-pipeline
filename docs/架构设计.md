# 系统架构设计

## 整体架构

```
┌──────────────────────────┐               ┌──────────────────────────────┐
│     MacBook Air M3       │   HTTP API    │    Windows 主机 (4070S)       │
│                          │ ────────────→ │                              │
│  ┌────────────────────┐  │   同一局域网   │  ┌────────────────────────┐  │
│  │  pipeline.py       │  │               │  │  GPT-SoVITS API        │  │
│  │  (主控脚本)         │  │               │  │  (api_v2.py :9880)     │  │
│  └────────┬───────────┘  │               │  └───────────┬────────────┘  │
│           │              │               │              │               │
│  ┌────────▼───────────┐  │               │  ┌───────────▼────────────┐  │
│  │  translate.py      │  │               │  │  音色模型 (GPT-SoVITS)  │  │
│  │  (LLM 翻译+增强)    │  │               │  │  - 零样本推理           │  │
│  └────────┬───────────┘  │               │  │  - Few-shot 微调模型    │  │
│           │              │               │  └────────────────────────┘  │
│  ┌────────▼───────────┐  │               │                              │
│  │  ffmpeg            │  │               │  ┌────────────────────────┐  │
│  │  (音频拼接)         │  │               │  │  参考音频素材库         │  │
│  └────────┬───────────┘  │               │  │  (buffett_ref_*.wav)   │  │
│           │              │               │  └────────────────────────┘  │
│  ┌────────▼───────────┐  │               │                              │
│  │  output/           │  │               └──────────────────────────────┘
│  │  (MP3 输出)        │  │
│  └────────────────────┘  │
│                          │
│  → AirDrop / iCloud → 手机│
└──────────────────────────┘
```

## 数据流

```
英文 transcript (input.txt)
      │
      ▼
[translate.py] ──LLM API──→ 中文翻译+背景增强 (translated.txt)
      │
      ▼
[pipeline.py] 分段 → segments[]
      │
      ├──段1──→ HTTP POST /tts ──→ Windows GPT-SoVITS ──→ seg_0001.wav
      ├──段2──→ HTTP POST /tts ──→ Windows GPT-SoVITS ──→ seg_0002.wav
      ├──...
      └──段N──→ HTTP POST /tts ──→ Windows GPT-SoVITS ──→ seg_000N.wav
                                                        │
                                                        ▼
                  ←──── 拉回 WAV 文件 ──────────────────┘
                        │
                        ▼
                  [ffmpeg 拼接 + 静音间隔]
                        │
                        ▼
                  output/buffett_qa_20250701.mp3
                        │
                        ▼
                  AirDrop → 手机
```

## 技术选型

| 层级 | 技术 | 版本 | 选型理由 |
|------|------|------|----------|
| LLM | DeepSeek API | deepseek-chat | 性价比高，中文翻译质量好 |
| TTS | GPT-SoVITS | v3 | 音色相似度 95%+，5s 零样本克隆，社区活跃 |
| 远程调用 | HTTP REST API | — | 简单可靠，GPT-SoVITS 内置 API 服务 |
| 音频处理 | ffmpeg | latest | 通用音频处理工具 |
| 编排 | Python | 3.10+ | 跨平台，生态丰富 |
| 运行环境 | Mac (M3) + Windows (4070S) | — | Mac 做轻量调度，Windows 做 GPU 推理 |

## GPT-SoVITS API 接口

### 请求

```
POST http://<windows-ip>:9880/tts
Content-Type: application/json

{
  "text": "要合成的中文文本",
  "text_lang": "zh",
  "ref_audio_path": "C:\\audio\\buffett_ref_10s.wav",
  "prompt_lang": "en",
  "top_k": 5,
  "top_p": 0.8,
  "temperature": 0.8,
  "text_split_method": "cut0",
  "speed_factor": 1.0
}
```

### 响应

```
binary audio/wav  →  直接保存为 .wav 文件
```
