# 巴菲特股东会音频流水线 — 部署指南

## 架构概览

```
┌──────────────────────┐          ┌──────────────────────────────┐
│   MacBook Air M3     │   SSH    │   Windows 主机 (RTX 4070S)   │
│                      │ ───────→ │                              │
│  🧠 LLM 翻译+增强     │          │  🎤 GPT-SoVITS 音色克隆      │
│  📋 流水线调度        │          │  🔊 批量 TTS 推理            │
│  📡 内容获取          │          │  🎛️  音频后处理              │
│  📱 推送到手机        │          │                              │
└──────────────────────┘          └──────────────────────────────┘
```

> 核心原则：**GPU 重活全在 Windows 上跑，Mac 做轻量任务和调度中枢。翻译用云端 API（不占本地资源），TTS 用 Windows GPU。**

## 第一步：Windows 主机环境搭建（约 30-45 分钟）

### 1.1 确认基础环境

打开 PowerShell 或 CMD，确认：

```powershell
# 检查 CUDA 驱动版本
nvidia-smi

# 期望输出：Driver Version >= 545.x, CUDA Version >= 12.4
```

如果没有 `nvidia-smi`，去 [NVIDIA 官网](https://www.nvidia.com/download/index.aspx) 下载对应 4070 Super 的 Game Ready 或 Studio 驱动。

### 1.2 安装 Python 3.10

推荐用 `conda`（Miniconda 即可）：

```powershell
# 下载并安装 Miniconda（如果还没有）
# https://docs.conda.io/en/latest/miniconda.html

# 创建 Python 3.10 环境
conda create -n gpt_sovits python=3.10 -y
conda activate gpt_sovits
```

### 1.3 安装 CUDA 版 PyTorch

```powershell
conda activate gpt_sovits
pip install torch torchaudio --index-url https://download.pytorch.org/whl/cu124
```

验证：
```powershell
python -c "import torch; print('CUDA:', torch.cuda.is_available()); print('GPU:', torch.cuda.get_device_name(0))"
# 期望输出：CUDA: True  GPU: NVIDIA GeForce RTX 4070 SUPER
```

### 1.4 安装 GPT-SoVITS

```powershell
# 克隆仓库
cd C:\
git clone https://github.com/RVC-Boss/GPT-SoVITS.git
cd GPT-SoVITS

# 安装依赖
conda activate gpt_sovits
pip install -r requirements.txt
```

### 1.5 下载预训练模型

两种方式：

**方式 A（推荐）：通过 WebUI 自动下载**
```powershell
python webui.py
# 首次启动会自动下载模型，约 3-5 GB
```

**方式 B：手动下载**
从 [GPT-SoVITS HuggingFace](https://huggingface.co/lj1995/GPT-SoVITS) 下载模型文件放到 `GPT_SoVITS/pretrained_models/`。

### 1.6 配置 SSH 远程访问（可选，方便 Mac 远程操作）

Windows 端：
```powershell

```

Mac 端测试连接：

## 第二步：收集巴菲特音频素材（Mac 上操作）

### 2.1 下载工具

```bash
# 安装 yt-dlp（YouTube 下载工具）
brew install yt-dlp ffmpeg

# 或者用 pip
pip3 install yt-dlp
```

### 2.2 推荐素材来源

| 来源 | 链接 | 特点 |
|------|------|------|
| CNBC Buffett Archive | https://buffett.cnbc.com/ | 最大巴菲特视频库 |
| 2025 股东会直播录像 | YouTube 搜索 "Berkshire Hathaway 2025 Annual Meeting" | 5+ 小时素材 |
| HBO 纪录片 | "Becoming Warren Buffett" | 高质量人声 |
| 经典采访 | YouTube 搜索 "Warren Buffett interview" | 丰富多样 |

### 2.3 下载 + 提取音频

```bash
# 下载一个视频并提取音频为 WAV（16kHz 单声道，GPT-SoVITS 推荐格式）
yt-dlp -f "bestaudio" --extract-audio --audio-format wav \
  --audio-quality 0 \
  --postprocessor-args "-ar 16000 -ac 1" \
  -o "buffett_%(title)s.%(ext)s" \
  "YouTube视频URL"
```

### 2.4 裁剪干净片段

用任意音频编辑工具（Audacity / GarageBand / ffmpeg）裁剪出 3-5 段干净的人声片段：

- 每段 **5-15 秒**
- 尽量选择 **纯人声、无背景音乐、无其他人说话** 的片段
- 保存为 16kHz 单声道 WAV

```bash
# ffmpeg 示例：从 1:30 开始裁剪 10 秒
ffmpeg -i input.wav -ss 00:01:30 -t 10 -ar 16000 -ac 1 buffett_clip_01.wav
```

## 第三步：巴菲特色音克隆训练（Windows 上操作）

### 3.1 启动 GPT-SoVITS WebUI

```powershell
conda activate gpt_sovits
cd C:\GPT-SoVITS
python webui.py
```

浏览器打开 `http://127.0.0.1:7860`（如果从 Mac 远程访问，用 `http://Windows的IP:7860`）

### 3.2 零样本推理（最快，先试效果）

1. 进入 **GPT-SoVITS-TTS** 标签页
2. 勾选 **"是否开启零样本 TTS"**
3. 上传一段巴菲特的 5-10 秒参考音频
4. 输入一段英文文本
5. 点击合成，听效果

### 3.3 Few-shot 微调（效果更好）

如果零样本效果不够好，做 few-shot 微调：

1. 准备 **1-3 分钟**的巴菲特干净人声（多段拼接即可）
2. 进入 **语音切分 & 特征提取** 标签页，上传音频
3. 自动切分 + ASR 识别生成标注数据集
4. 进入 **模型微调** 标签页，使用默认参数训练
5. 训练完成（4070S 上约 30-60 分钟），推理质量明显提升

### 3.4 调整音色参数

GPT-SoVITS 支持调整音色特征：
- **top_k / top_p**：控制音色变化的随机性（越小越稳定）
- **语速**：可调整为稍慢（巴菲特本身语速偏慢）
- **音高**：可微调使声音更清晰（+1 到 +3 个半音可使老年音更清晰）

## 第四步：翻译 + 背景增强流水线（Mac 上操作）

### 4.1 获取股东会英文原文

优质来源：
- https://steadycompounding.com/transcript/brk-2025/ （2025 年完整 transcript）
- https://buffett.cnbc.com/annual-meetings/ （CNBC 官方历年记录）
- 雪球/知乎上用户整理的英文版

### 4.2 LLM 翻译脚本

```python
# translate_and_enhance.py
import requests
import json

API_KEY = "你的 DeepSeek API Key"  # 或 OpenAI API Key
API_URL = "https://api.deepseek.com/chat/completions"

SYSTEM_PROMPT = """你是一位精通巴菲特投资哲学的翻译专家。你的任务是：

1. 将伯克希尔·哈撒韦股东大会的英文问答翻译成地道的中文
2. 在翻译中，当涉及以下内容时，用【背景补充：...】的方式插入背景信息：
   - 提到的公司当时的基本面情况（市值、股价、PE等）
   - 当时的宏观经济背景（利率、通胀、市场环境等）
   - 伯克希尔当时的持仓情况和仓位变化
   - 巴菲特/Munger 过往对该问题的相关论述
   - 事件后续发展（如果已知的话）
3. 翻译风格：口语化、自然流畅，保留巴菲特的幽默感和智慧
4. 不要过度补充背景信息，每段问答最多 1-2 处背景补充

格式：
【提问人】XXX（来自XXX机构）
【问题】...中文翻译...
【巴菲特回答】...中文翻译...
【背景补充】...（如果有的话）...
---
"""

def translate_qa(english_text: str) -> str:
    response = requests.post(
        API_URL,
        headers={"Authorization": f"Bearer {API_KEY}"},
        json={
            "model": "deepseek-chat",
            "messages": [
                {"role": "system", "content": SYSTEM_PROMPT},
                {"role": "user", "content": f"请翻译以下股东会问答：\n\n{english_text}"}
            ],
            "temperature": 0.7,
        }
    )
    return response.json()["choices"][0]["message"]["content"]

# 使用示例
with open("brk_2025_qa_sample.txt", "r") as f:
    text = f.read()

result = translate_qa(text)
print(result)

# 保存结果
with open("brk_2025_qa_cn.json", "w") as f:
    json.dump({"translated": result}, f, ensure_ascii=False, indent=2)
```

## 推荐的操作流程（日常使用）

```
每次想生成新内容时：

1. Mac 上：找到新的股东会 transcript
           ↓
2. Mac 上：运行 translate.py → 翻译+增强
           ↓
3. Mac 上：运行 pipeline.py → 自动远程调用 Windows TTS
           ↓
4. Windows 自动完成合成，音频文件回传 Mac
           ↓
5. Mac 上：ffmpeg 拼接 + 上传到手机
           ↓
6. 手机上：用播客 App 或文件管理器收听
```
